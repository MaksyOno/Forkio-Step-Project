2) Использованные технологии:
- сборщик проекта Gulp, со следующими модулями:
	- sass (установка предпроцессора);
	- clean (для очистки сборочной директории);
	- cleanCSS (сжатие и оптимизация CSS файлов);
	- sourcemaps (для навигации по файлам проекта и отображения их местонахождения);
	- concat (для объединения нескольких файлов в один);
	- rename (для переименования объединенных файлов);
	- terser (сжатие и оптимизация JS файлов)
	- autoprefixer (для расставки вендорных префиксов);
	- imagemin (для оптимизации и сжатия изображений);
	- browser-sync (автоматическм перезагрузка страници, после внесения изменений);

- предпроцессоры SCSS; (для настройки удобной структуры проекта);
